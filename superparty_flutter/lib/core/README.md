# Core

Infrastructură centrală a aplicației, independentă de feature-uri specifice.

## Structură

- `di/` - Dependency Injection (get_it setup, registrări)
- `routing/` - Routing configuration (go_router)
- `errors/` - Error handling (Failure types, Result/Either)
- `logging/` - Logging centralizat
- `utils/` - Utilitare generale (validators, formatters, etc.)
